## 📈 机场评测标准

我们根据以下标准对机场进行评测：

1. **速度测试** (30%)
   - 下载速度
   - 上传速度
   - 延迟测试

2. **稳定性** (25%)
   - 连接成功率
   - 断线频率
   - 服务可用性

3. **节点质量** (20%)
   - 节点数量
   - 地区分布
   - IP质量

4. **解锁能力** (15%)
   - 流媒体解锁
   - 游戏支持
   - 特殊服务

5. **用户体验** (10%)
   - 客服响应
   - 界面友好度
   - 使用便利性
