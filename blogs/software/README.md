## 使用建议

::: tip 软件选择建议
1. **代理客户端**: 根据平台选择对应客户端，推荐Clash系列
2. **浏览器**: 建议使用Chrome或Firefox，注意隐私设置
3. **下载工具**: Windows推荐IDM，其他平台可选择FDM
4. **安全软件**: 定期使用Malwarebytes扫描系统
:::

::: warning 安全提醒
- 请从官方渠道下载软件，避免使用破解版
- 定期更新软件版本，确保安全性
- 注意软件权限设置，保护个人隐私
:::
